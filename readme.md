React with Server Side Rendering :

Tools :
1. Yarn version >= 1.6.0
2. Node version >= 8.X.X

Technology :
1. React 16.4
2. React Router 4
3. Webpack 4
4. Jest, Enzmy
5. Styled Component
6. Prettier
7. Express

run `yarn install` first to install depedencies.

Running :
`yarn start`

running at http://localhost:8000/
